# New Engineer Onboarding

Welcome to Enterprise Demo. This is the short version.

## Day 1

- Pair with the on-call engineer for the first deploy of the day.
- Read `CLAUDE.md` end-to-end — it documents the CLI discovery workflow and
  the read-only `core/` submodule rule.
- Run `./demo/00-preflight.sh && ./demo/01-seed-data.sh` to seed your local
  profile with governance history, agents, skills, and sample content.

## Week 1

- Walk through every demo under `demo/` to build an end-to-end mental model
  of the platform.
- File one PR, however small, against this template repo and land it through
  the review + sweep cycle.
- Attend a retro.

## Month 1

- Ship a governance rule extension or a new plugin skill.
- Own an on-call rotation.
